#column-gap

Определяет зазор между столбцами элемента.

column-gap: normal;

Расстояние между столбцами устанавливается в соответствии со значением браузера по умолчанию, которое обычно составляет 1em.

Hello world

Foo bar

CSS Reference

column-gap: 2px;

Для зазора можно использовать значения пикселей.

Обратите внимание, что зазор появляется только между колоннами, а не на внешних сторонах крайних колонн.
