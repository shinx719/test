#column-count

Определяет количество столбцов элемента.

column-count: auto;

Удаляет все столбцы из элемента (если не column-было задано другое свойство).

Hello world

Foo bar

CSS Reference

column-count: 3;

При использовании целочисленного значения элемент распределит свои дочерние элементы по количеству определенных столбцов.

Hello world

Foo bar

CSS Reference
