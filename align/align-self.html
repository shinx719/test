#align-self

Работает как align-items, но применяется только к одному элементу flexbox, а не ко всем сразу.

align-self: auto;

Целевой элемент будет использовать значение align-items.

align-self: flex-start;

Если у контейнера установлено align-items: center, а у целевого элемента — align-self: flex-start, только целевой элемент будет находиться в начале поперечной оси.

По умолчанию это означает, что элемент будет выровнен вертикально по верхнему краю.

align-self: flex-end;

Если у контейнера установлено align-items: center, а у целевого элемента — align-self: flex-end, только целевой элемент будет находиться в конце поперечной оси.

По умолчанию это означает, что элемент будет выровнен вертикально по нижнему краю.

align-self: center;

Если у контейнера установлено align-items: flex-start, а у целевого элемента — align-self: center, только целевой элемент будет находиться в центре поперечной оси.

По умолчанию это означает, что элемент будет выровнен по вертикали по центру.

align-self: baseline;

Если у контейнера установлено align-items: center, а у целевого элемента — align-self: baseline, только целевой элемент будет находиться на базовой линии поперечной оси.

По умолчанию это означает, что элемент будет выровнен вдоль базовой линии текста.
