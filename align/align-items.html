#align-items

align-items: flex-start;

Определяет, как элементы flexbox выравниваются по поперечной оси внутри линии контейнера flexbox.

По умолчанию поперечная ось является вертикальной. Это означает, что элементы flexbox будут выравниваться вертикально по верхнему краю.

align-items: flex-end;

Элементы flexbox выравниваются по концу поперечной оси.

По умолчанию поперечная ось является вертикальной. Это означает, что элементы flexbox будут выравниваться вертикально по нижнему краю.

align-items: center;

Элементы flexbox выравниваются по центру поперечной оси.

По умолчанию поперечная ось является вертикальной. Это означает, что элементы flexbox будут выравниваться по центру вертикально.

align-items: baseline;

Элементы flexbox выравниваются по базовой линии поперечной оси.

По умолчанию поперечная ось является вертикальной. Это означает, что элементы flexbox выровняются так, чтобы базовые линии их текста располагались по одной горизонтальной линии.

align-items: stretch;

Элементы flexbox будут растягиваться по всей поперечной оси.

По умолчанию поперечная ось является вертикальной. Это означает, что элементы flexbox заполнят все вертикальное пространство.
