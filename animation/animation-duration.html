animation-duration

Определяет, сколько времени будет длиться анимация

animation-duration: 0s;

Обычное значение ноль: анимация не будет проигрываться.

0s

animation-duration: 1.2s;

Вы можете использовать десятичные значения в секундах с ключевым словом s.

1.2s

animation-duration: 2400ms;

Вы можете использовать миллисекунды вместо секунд, с ключевым словом ms.

2400ms
