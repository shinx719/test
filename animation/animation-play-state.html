animation-play-state

Определяет будет ли проигрываться анимация

animation-play-state: running;

Если заданы animation-duration и animation-name, анимация начнёт воспроизводиться автоматически.

none

animation-play-state: paused;

Анимация установлена в состояние "приостановлено" на первом ключевом кадре.

Это отличается от ситуации, когда не указаны animation-duration или animation-name вообще. Если анимация приостановлена, применяется стиль первого ключевого кадра, а не стиль по умолчанию.

В этом примере квадрат видим по умолчанию, но на первом ключевом кадре анимации fadeAndMove его непрозрачность (opacity) установлена в 0. Когда анимация приостановлена, она будет "застрявшей" на этом первом ключевом кадре, и, следовательно, квадрат будет невидим. 

                @keyframes fadeAndMove {
                    from {
                        opacity: 0;
                        transform: translateX(0);
                    }
                    to {
                        opacity: 0;
                        transform: translateX(100px);
                    }
                    }
