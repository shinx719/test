animation-name

Определяет, какие ключевые кадры анимации использовать.

animation-name: none;

Если имя анимации не указано, анимация не воспроизводится.

none

animation-name: fadeIn;

Если указано имя, будут использованы ключевые кадры, соответствующие этому имени. Например, анимация fadeIn выглядит так: 

                @keyframes fadeIn {
                    from {
                      opacity: 0;
                    }
                    to {
                      opacity: 1;
                    }
                  }

fadeIn

animation-name: moveRight;

Другой пример: анимация moveRight: 

                @keyframes moveRight {
                    from {
                      transform: translateX(0);
                    }
                    to {
                      transform: translateX(100px);
                    }
                  }

moveRight
