#animation-iteration-count

Определяет сколько раз проиграется анимация

animation-iteration-count: 1;

Анимация проиграется только один раз

none

animation-iteration-count: 2;

Можно использовать целые числа для выбора сколько раз проиграется анимация

none

animation-iteration-count: infinite;

Используя infinite анимация будет проигрываться бесконечно

none
